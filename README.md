## 🛠️ How to Use

1. **Clone this repo**  
   ```bash
   git clone https://github.com/shoryaagrawal553/DSA.git
   cd DSA
   ```

2. **Compile and run a C++ file**  
   ```bash
   g++ path/to/your/file.cpp -o output
   ./output
   ```

3. **Browse and learn**  
   Dive into a folder, pick a problem, and level up your skills! 🚀
